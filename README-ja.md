# VueDraggableSample

## 概要

D＆D で親子関係のタスクを移動するサンプルです。
各ステータスをグループ分けして、グループごとの移動制限しています。

## 環境

- Vue 3.5.13
- Vite v6.2.5
- NPM 10.1.0

## プロジェクトセットアップ

### パッケージインストール

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
